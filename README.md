# Система онлайн-записи на консультации (Демо)

Учебный демо-сайт для записи студентов на консультации преподавателей.

## Что реализовано

- Авторизация по логину/паролю из отдельного файла базы `data/demo-db.json`.
- Роли:
  - `student`: просмотр свободных консультаций, запись, отмена, просмотр своих записей.
  - `teacher`: добавление слотов консультаций, перенос, отмена, просмотр записавшихся студентов.
  - `admin`: добавление и удаление пользователей, редактирование преподавателей, обзор по расписанию и записям.
- Минималистичный адаптивный интерфейс в образовательной тематике.
- Демо-сохранение данных через `localStorage` (без реальной серверной БД).

## Структура проекта

- `index.html` - страница приложения.
- `styles.css` - стили.
- `app.js` - логика приложения.
- `data/demo-db.json` - отдельный файл "базы данных" с демо-пользователями и расписанием.

## Запуск

Важно запускать через локальный сервер, чтобы работал `fetch` JSON-файла.

Пример на Python:

```bash
python -m http.server 5500
```

Затем откройте:

`http://localhost:5500`

## Демо-аккаунты

- Студент: `student1` / `student123`
- Преподаватель: `teacher1` / `teacher123`
- Администратор: `admin` / `admin123`

## Примечание

Это учебный демо-проект: пароли хранятся в открытом JSON-файле и не предназначены для реального продакшн-использования.
